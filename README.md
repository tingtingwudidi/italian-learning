<a name="english"></a>
## 🇺🇸 English Description
A comprehensive Italian learning platform covering 9 core life scenarios, using visual and auditory cues to help beginners master essential vocabulary.

* **Scenario-based**: Lessons include "First Meetings," "Numbers," "Dining," and more.
* **Voice Synthesis (TTS)**: Built-in Italian pronunciation engine for every word and example.
* **iOS Optimized**: Custom "Add to Home Screen" prompt for a seamless full-screen experience on iPhone.
* **Eye-care Design**: Dark green theme to reduce eye strain during long study sessions.

<a name="chinese"></a>
## 🇨🇳 中文说明
涵盖 9 大核心生活场景的意大利语入门平台，通过视听结合帮助初学者快速上手。

* **场景化学习**：包括“初次见面”、“数字”、“餐厅饮食”等实用分类。
* **语音合成 (TTS)**：点击单词或例句即可聆听地道的意大利语发音。
* **iOS 深度优化**：内置“添加到主屏幕”引导，提供沉浸式 App 体验。
* **护眼设计**：采用深色调墨绿主题，适合长时间专注学习。
